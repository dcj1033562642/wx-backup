# wx-backup
##微x配置备份
这是我的第一个GitHub，用于复活微x模块
##如何使用
1.下载配置包，然后先降级微信到8.0.50已下
2.解压后打开，把FKZ_WX_DATA和x7_3.0复制到/storage/emulated/0/Android/data/com.tencent.mm/
3.安装微x模块和修复模块2.0，并开启模块
4.强行停止微信再打开，点击弹窗左下角的更多
5.导入FKZ_WX_DATA，粘贴/storage/emulated/0/Android/data/com.tencent.mm/FKZ_WX_DATA
6.导入核心文件粘贴/storage/emulated/0/Android/data/com.tencent.mm/x7_3.0
7.重启微信即可

特别说明：配置包写的哪个微信版本，就安装哪个版本
